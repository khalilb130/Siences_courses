## 📖 À propos du projet

**Sciences Courses** est une collection de **cartes mentales interactives** conçues pour aider les lycéens à maîtriser les concepts scientifiques complexes. Chaque carte combine :

- 🎨 **Design moderne** avec animations et interactions
- 📝 **Contenu complet** : théorie, formules, méthodes et exercices
- 💡 **Astuces pratiques** pour réussir les contrôles
- 🎯 **Organisation claire** pour des révisions efficaces

> *"Apprendre en s'amusant, réviser efficacement, réussir brillamment !"*

### ✅ Chapitres disponibles

<details open>
<summary><b>📐 Produit Scalaire</b> 🆕</summary>

#### Contenu complet :
- **6 sections interactives** : 
  - Définitions (analytique, géométrique, projection)
  - Propriétés (commutativité, distributivité, bilinéarité)
  - Orthogonalité (tests et applications)
  - Applications géométriques (Al-Kashi, médiane, hauteur)
  - Équations de cercle et tangentes
  - Calculs d'angles

#### Bonus :
- ✨ **Relation de Chasles** avec exemple détaillé
- 📝 **Exercice type** : Carré avec solution en 2 méthodes
- 🎯 **Carte globale** : 5 méthodes pour réussir le DS
- 💡 **6 astuces** pour éviter les pièges

</details>

<details open>
<summary><b>🔢 Suites Arithmético-Géométriques</b> 🆕</summary>

#### Contenu complet :
- **6 sections interactives** :
  - Définition et reconnaissance (uₙ₊₁ = a·uₙ + b)
  - Point fixe ℓ = b/(1-a)
  - Suite auxiliaire géométrique
  - Formule explicite : uₙ = (u₀ - ℓ)aⁿ + ℓ
  - Méthode de résolution en 4 étapes

#### Bonus :

### 📥 Installation locale

```bash
# 1. Cloner le dépôt
```

## 🎓 Pour les étudiants

### 🎯 Objectifs pédagogiques

Ces cartes mentales vous permettent de :

| Objectif | Avantage |
|:---|:---|
| ✅ **Comprendre** | Explications claires et visuelles |
| ✅ **Mémoriser** | Organisation logique et hiérarchique |
| ✅ **Réviser** | Accès rapide aux notions essentielles |
| ✅ **Pratiquer** | Exercices types avec corrections détaillées |
| ✅ **Réussir** | Méthodes éprouvées et astuces anti-pièges |

### 💡 Comment utiliser ces cartes ?

1. **Avant le cours** : Découvrir les notions
2. **Après le cours** : Approfondir et comprendre
3. **Avant un contrôle** : Réviser l'essentiel
4. **Pendant les révisions** : Synthétiser et mémoriser
