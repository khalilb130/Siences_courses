# 📚 Sciences Courses - Cartes Mentales Interactives

Bienvenue sur le dépôt de **cartes mentales interactives** pour les sciences au lycée !

## 🎯 À propos

Ce projet propose des cartes mentales modernes et interactives pour faciliter l'apprentissage et la révision des cours de sciences (Mathématiques, Physique-Chimie) au niveau lycée.

## 🌐 Accès en ligne

**🔗 Site web :** [https://votre-username.github.io/Siences_courses/](https://votre-username.github.io/Siences_courses/)

*(Remplacez `votre-username` par votre nom d'utilisateur GitHub)*

## 📐 Mathématiques - Niveau Première

### ✅ Chapitres disponibles

1. **📐 Produit Scalaire**
   - 6 sections interactives : Définitions, Propriétés, Orthogonalité, Applications géométriques, Équations de cercle, Calculs d'angles
   - Relation de Chasles avec exemple détaillé
   - Exercices types corrigés
   - Carte globale avec méthodes pour réussir le DS

2. **🔢 Suites Arithmético-Géométriques**
   - 6 sections interactives : Définition, Point fixe, Suite auxiliaire, Formule explicite, Méthode de résolution, Comportement
   - Méthode complète en 4 étapes
   - Astuces et pièges à éviter
   - Carte globale avec exercices types

## ✨ Fonctionnalités

- 🎨 **Interface moderne et intuitive** : Design coloré avec animations fluides
- 🖱️ **Navigation interactive** : Cliquez sur chaque section pour explorer
- 📋 **Carte globale** : Cliquez sur la bulle centrale pour accéder aux notions incontournables et méthodes DS
- 📝 **Exercices types** : Exemples détaillés et corrigés pas à pas
- 💡 **Astuces et pièges** : Conseils pour réussir et erreurs à éviter
- 📱 **Responsive** : Fonctionne sur ordinateur, tablette et smartphone
- 🚀 **Révisions efficaces** : Organisation claire pour des révisions rapides

## 🛠️ Technologies utilisées

- HTML5
- CSS3 (animations, flexbox, grid)
- JavaScript (interactivité)
- Design responsive
