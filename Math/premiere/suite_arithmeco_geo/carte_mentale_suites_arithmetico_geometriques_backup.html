Notions incontournables & Méthodes pour réussir ton DS

🎯 LES NOTIONS INCONTOURNABLES

1. Définition et reconnaissance

uₙ₊₁ = a·uₙ + b
  • avec a ≠ 1 et b ≠ 0
  • Si a = 1 → suite arithmétique
  • Si b = 0 → suite géométrique
💡 Bien identifier les valeurs de a, b et u₀

2. Point fixe ℓ

ℓ = b / (1 - a)
  • C'est la solution de : ℓ = a·ℓ + b
  • Si la suite converge, elle converge vers ℓ
💡 Toujours calculer ℓ en premier !

3. Suite auxiliaire

vₙ = uₙ - ℓ
  • (vₙ) est une suite géométrique de raison a
  • vₙ₊₁ = a·vₙ
  • vₙ = v₀ × aⁿ = (u₀ - ℓ) × aⁿ

4. Formule explicite ⭐

uₙ = (u₀ - ℓ) × aⁿ + ℓ
🔥 LA FORMULE À CONNAÎTRE PAR CŒUR !

5. Comportement selon a

  • |a| < 1 : la suite converge vers ℓ
  • a > 1 : la suite diverge (vers +∞ ou -∞)
  • a < -1 : la suite diverge en oscillant
  • a = -1 : la suite oscille entre deux valeurs
💡 Retenir : convergence ssi -1 < a < 1

📝 MÉTHODES POUR RÉUSSIR TON DS

Méthode 1 : Trouver la formule explicite de uₙ

  • ÉTAPE 1 : Calculer le point fixe ℓ = b/(1-a)
  • ÉTAPE 2 : Poser vₙ = uₙ - ℓ
  • ÉTAPE 3 : Montrer que vₙ₊₁ = a·vₙ (démonstration)
  • ÉTAPE 4 : En déduire vₙ = (u₀ - ℓ) × aⁿ
  • ÉTAPE 5 : Conclure : uₙ = vₙ + ℓ = (u₀ - ℓ)aⁿ + ℓ
  • ÉTAPE 6 : Vérifier avec u₀ et u₁
⚠️ Ne JAMAIS oublier le "+ ℓ" à la fin !

Méthode 2 : Calculer un terme de la suite

  • Si formule explicite trouvée : utiliser uₙ = (u₀ - ℓ)aⁿ + ℓ
  • Sinon, utiliser la récurrence : uₙ₊₁ = a·uₙ + b
  • Calculer terme par terme jusqu'au rang demandé
  • Vérifier la cohérence des résultats

Méthode 3 : Étudier la convergence

  • Calculer le point fixe ℓ
  • Déterminer la valeur de a
  • Si -1 < a < 1 : la suite converge vers ℓ
  • Sinon : la suite diverge
  • Justifier : "aⁿ → 0 donc uₙ → ℓ" ou "aⁿ → ±∞"

Méthode 4 : Démontrer que vₙ est géométrique

  • Calculer vₙ₊₁ en fonction de uₙ₊₁
  • Remplacer uₙ₊₁ par a·uₙ + b
  • Factoriser pour faire apparaître (uₙ - ℓ)
  • Utiliser le fait que ℓ = a·ℓ + b
  • Conclure : vₙ₊₁ = a·vₙ donc (vₙ) est géométrique de raison a

Méthode 5 : Étudier la monotonie

  • Calculer uₙ₊₁ - uₙ
  • OU étudier le signe de uₙ₊₁ - uₙ = (a-1)uₙ + b
  • OU comparer uₙ avec ℓ selon que u₀ > ℓ ou u₀ < ℓ
  • Attention : si a < 0, la suite n'est ni croissante ni décroissante

Méthode 6 : Résoudre une inéquation uₙ > k

  • Utiliser la formule explicite : (u₀ - ℓ)aⁿ + ℓ > k
  • Isoler aⁿ : aⁿ > (k - ℓ)/(u₀ - ℓ)
  • Prendre le logarithme (attention au sens selon a)
  • Résoudre pour n

⚡ ASTUCES POUR LE DS

✓ Mémoriser la formule magique : uₙ = (u₀ - ℓ)aⁿ + ℓ
✓ Calculer ℓ dès le départ : c'est la clé de tout l'exercice
✓ Vérifier avec u₀ et u₁ : ta formule doit donner les bonnes valeurs
✓ Ne pas confondre a et b : bien identifier uₙ₊₁ = a·uₙ + b
✓ Faire apparaître (uₙ - ℓ) : c'est le réflexe pour montrer que vₙ est géométrique
✓ Attention au signe de a : détermine le comportement de la suite
✓ Convergence = |a| < 1 : sinon divergence !
✓ Calcul de u₀ - ℓ : ne pas oublier les parenthèses si négatif !

🚨 PIÈGES À ÉVITER

❌ Piège 1 : Oublier le "+ ℓ" dans la formule finale
❌ Piège 2 : Confondre a et (1-a) dans le calcul de ℓ
❌ Piège 3 : Dire qu'une suite converge sans vérifier |a| < 1
❌ Piège 4 : Se tromper dans le calcul de (u₀ - ℓ)
❌ Piège 5 : Oublier de justifier que vₙ est géométrique

📚 Définition

Suite arithmético-géométrique :

Une suite (uₙ) définie par une relation de récurrence de la forme :

uₙ₊₁ = a·uₙ + b

où a ≠ 1 et b ≠ 0

Exemples :
• uₙ₊₁ = 2uₙ + 3, u₀ = 1
• uₙ₊₁ = 0,5uₙ + 4, u₀ = 10
• uₙ₊₁ = -3uₙ + 7, u₀ = 2

Origine du nom :

Combine les propriétés :

  • Géométrique : multiplication par a
  • Arithmétique : ajout de b

Cas exclus :

  • Si a = 1 : suite arithmétique (uₙ₊₁ = uₙ + b)
  • Si b = 0 : suite géométrique (uₙ₊₁ = a·uₙ)

🎯 Point Fixe

Définition du point fixe ℓ :

Le point fixe est la solution de :

ℓ = a·ℓ + b

Résolution :

ℓ = b / (1 - a)

(avec a ≠ 1)

Exemple 1 :
uₙ₊₁ = 2uₙ + 3
ℓ = 2ℓ + 3
ℓ - 2ℓ = 3
-ℓ = 3
ℓ = -3
Exemple 2 :
uₙ₊₁ = 0,5uₙ + 4
ℓ = 0,5ℓ + 4
0,5ℓ = 4
ℓ = 8

Interprétation :

Si la suite converge, elle converge vers ℓ

🔄 Suite Auxiliaire

Méthode de transformation :

On pose la suite auxiliaire :

vₙ = uₙ - ℓ

Propriété fondamentale :

(vₙ) est une suite géométrique de raison a

vₙ₊₁ = a·vₙ
Démonstration :
vₙ₊₁ = uₙ₊₁ - ℓ
= (a·uₙ + b) - ℓ
= a·uₙ + b - (a·ℓ + b)
= a·uₙ - a·ℓ
= a(uₙ - ℓ)
= a·vₙ

Terme général de (vₙ) :

vₙ = v₀ × aⁿ = (u₀ - ℓ) × aⁿ

📐 Formule Explicite

Formule du terme général :

uₙ = (u₀ - ℓ) × aⁿ + ℓ

Dérivation :

Puisque vₙ = uₙ - ℓ, alors :

uₙ = vₙ + ℓ = (u₀ - ℓ)aⁿ + ℓ
Exemple complet :
uₙ₊₁ = 2uₙ + 3, u₀ = 1

Étape 1 : Point fixe
ℓ = 2ℓ + 3 → ℓ = -3

Étape 2 : Formule
uₙ = (1 - (-3)) × 2ⁿ + (-3)
uₙ = 4 × 2ⁿ - 3

Vérification :
u₀ = 4×1 - 3 = 1 ✓
u₁ = 4×2 - 3 = 5 ✓
u₂ = 4×4 - 3 = 13 ✓
Autre exemple :
uₙ₊₁ = 0,5uₙ + 4, u₀ = 10

Point fixe : ℓ = 8
Formule : uₙ = (10 - 8) × 0,5ⁿ + 8
uₙ = 2 × 0,5ⁿ + 8

🛠️ Méthode de Résolution

Les 4 étapes :

ÉTAPE 1 : Trouver le point fixe
Résoudre : ℓ = a·ℓ + b
→ ℓ = b/(1-a)
ÉTAPE 2 : Définir la suite auxiliaire
Poser : vₙ = uₙ - ℓ
Montrer que vₙ₊₁ = a·vₙ
ÉTAPE 3 : Trouver vₙ
(vₙ) est géométrique de raison a
→ vₙ = v₀ × aⁿ = (u₀ - ℓ) × aⁿ
ÉTAPE 4 : Déduire uₙ
uₙ = vₙ + ℓ
→ uₙ = (u₀ - ℓ)aⁿ + ℓ

⚠️ Points d'attention :

  • Vérifier que a ≠ 1 et b ≠ 0
  • Calculer correctement (u₀ - ℓ)
  • Ne pas oublier le "+ ℓ" final
  • Vérifier avec les premiers termes

📊 Comportement & Convergence

Selon la valeur de a :

Si |a| < 1 :
• aⁿ → 0 quand n → +∞
• (uₙ) converge vers ℓ
• lim uₙ = ℓ
Exemple : uₙ₊₁ = 0,5uₙ + 4
a = 0,5, ℓ = 8
La suite converge vers 8
Si a > 1 :
• aⁿ → +∞
• Si u₀ > ℓ : uₙ → +∞
• Si u₀ < ℓ : uₙ → -∞
Si a < -1 :
• aⁿ oscille et croît en valeur absolue
• La suite diverge (pas de limite)
Si a = -1 :
• La suite oscille entre deux valeurs
• Pas de limite (sauf si u₀ = ℓ)

Monotonie :

  • Si 0 < a < 1 et u₀ > ℓ : décroissante vers ℓ
  • Si 0 < a < 1 et u₀ < ℓ : croissante vers ℓ
  • Si a > 1 : monotone (croissante ou décroissante)
  • Si a < 0 : ni croissante ni décroissante
Résumé pratique :
• -1 < a < 1 : converge vers ℓ
• a > 1 ou a < -1 : diverge
• a = 1 : arithmétique
• a = -1 : oscille
