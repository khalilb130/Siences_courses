📘Énergie Cinétique

Ec = ½ m v²
  • ✓ Énergie du mouvement
  • ✓ Toujours positive ou nulle
  • ✓ Si v double → Ec × 4

🔧Théorème Ec (TEC)

ΔEc = W(forces)
  • ✓ Variation Ec = Travail total
  • ✓ Calcul de vitesse finale
  • ✓ Distance de freinage

📗Énergie Mécanique

Em = Ec + Epp
  • ✓ Epp = m g h
  • ✓ Dépend du niveau de référence
  • ✓ Conversion Ec ⇄ Epp

⚖️Conservation Em

Em(A) = Em(B)
  • ✓ Si forces conservatives uniquement
  • ✓ Pas de frottements
  • ✓ Chute libre, pendule idéal

🔴Variation Em

ΔEm = W(frottements)
  • ✓ Forces non conservatives
  • ✓ W < 0 → perte d'énergie
  • ✓ Dissipation en chaleur

💼Travail d'une Force

W = F · d · cos(α)
  • ✓ α = 0° → Moteur (W > 0)
  • ✓ α = 90° → Nul (W = 0)
  • ✓ α = 180° → Résistant (W < 0)
